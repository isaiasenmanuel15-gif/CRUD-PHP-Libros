# CRUD-PHP-Libros
"Actividad 3.1: Sistema de gestión de libros en PHP y MySQL

Proyecto: Sistema de Gestión de Libros (CRUD)
Este proyecto es un sistema funcional desarrollado en un solo archivo PHP.

Requisitos cumplidos:
* Base de Datos: MySQL con tabla de 5 campos (id, titulo, autor, anio, genero).
* Operaciones: Registro, visualización en tabla, edición y eliminación.
* Tecnologías: Uso de PHP, HTML, CSS básico y métodos GET/POST.
* Validaciones: Mensajes de confirmación y campos obligatorios.
